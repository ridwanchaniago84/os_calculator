# OS Calculator
This web is for calculating CPU Scheduling Algorithms, the existing algorithm is:
- FCFS
- SJF (Non preemptive)
- SJF (Preemptive)
- Priority
- Round-Robin Scheduling
# How to run
Just open the index.html with your browser.
